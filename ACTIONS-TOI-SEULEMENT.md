# Actions que toi seul peux faire (comptes externes)

## 1. GitHub
1. Créer un repo vide (ex: `albatros-studio-site`).
2. Pousser ce dossier.

## 2. Vercel
1. Importer le repo GitHub.
2. Déployer en production.
3. Attacher les domaines:
   - `albatros.studio`
   - `www.albatros.studio`

## 3. DNS + email forwarding
1. Ouvrir ton DNS (si domaine géré chez Vercel: Vercel Domains).
2. Ajouter les 2 enregistrements MX + 1 TXT ImprovMX.
3. Dans ImprovMX, créer alias:
   - `contact` -> `robinkliphuis@gmail.com`

## 4. Google for Nonprofits / Workspace
1. Soumettre la demande Google for Nonprofits.
2. Répondre à Goodstack si demandé.
3. Activer Google Workspace for Nonprofits.
4. Ajouter un TXT de vérification Google si demandé.
